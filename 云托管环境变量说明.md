# 微信云托管 - 环境变量配置

在云托管控制台为 **parking-share** 服务添加以下环境变量后，请**重新发布**一次服务使配置生效。

## 数据库（必填）

| 变量名 | 值 | 说明 |
|--------|-----|------|
| `SPRING_DATASOURCE_URL` | `jdbc:mysql://10.21.112.147:3306/parking_share?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true` | 内网数据库地址，库名 parking_share |
| `SPRING_DATASOURCE_USERNAME` | `root` | 数据库用户名 |
| `SPRING_DATASOURCE_PASSWORD` | `mima` | 数据库密码（请你在控制台改为自己的密码后，在此处同步修改） |

## 配置步骤

1. 打开 **微信云托管** → **服务列表** → 点击 **parking-share**。
2. 进入 **服务设置**（或 **部署发布** 里的 **高级设置**），找到 **环境变量**。
3. 新增上述 3 个变量，保存。
4. 回到 **部署发布**，点击 **发布**，等待新版本部署完成。

## 修改密码后

若你在数据库控制台把 root 密码从 `mima` 改成了其他密码，请把环境变量 `SPRING_DATASOURCE_PASSWORD` 改为新密码，并再次发布服务。
